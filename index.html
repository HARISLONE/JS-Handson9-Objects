Q.4): Complete the function in the editor. In which you are given an object as a parameter. You have to return "false" if there is no parameter in the object otherwise return "true".

Ans: 
             
                function Check(obj1) {
   
                    for(var prop in obj1) {
                     if (Object.prototype.hasOwnProperty.call(obj1, prop)) {
                       return "true";
                     }
                   }
                   return "false";
                    
                 }
